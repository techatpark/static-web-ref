The Behavior: JavaScript

JavaScript adds interactivity and dynamic behavior to the page. It handles actions like clicks, form submissions, or animations, making the page come alive. If HTML is the skeleton and CSS is the appearance, JavaScript is the movement and response.

    
    document.querySelector("h1").addEventListener("click", () => {
        alert("You clicked the heading!");
    });

With HTML, CSS, and JavaScript combined, the page is structured, styled, and interactive.
