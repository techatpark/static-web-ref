The Content: HTML

HTML (HyperText Markup Language) provides the structure of a webpage. It defines the content, such as headings, paragraphs, and images. Think of HTML as the skeleton — it sets up the foundation but doesn’t specify how things should look or behave.

        
        <h1>Welcome!</h1>
        <p>This is a basic web page.</p>
