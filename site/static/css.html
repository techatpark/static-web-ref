The Presentation: CSS

CSS (Cascading Style Sheets) is used to style the HTML content. It defines how elements look — their colors, sizes, positions, and fonts. If HTML is the skeleton, CSS is the clothes, makeup, and overall appearance.

    
    h1 {
        color: blue;
        text-align: center;
    }
    p {
        font-size: 18px;
        color: gray;
    }
